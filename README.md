CH is a hoodie customization platform created as a school project.

Users can design their own hoodies, upload their designs, and earn profit when other users buy their designs.

---

## Business Model

- Hoodie base price: **$100 (simulation money)**
- Upload subscription: **$5 per month (simulation money)**
- Designers earn profit when their hoodie is purchased
- All currency used is **not real money**

---

## Features

- Hoodie customization tool
- Draw letters only on a black hoodie
- Upload images to hoodie designs
- Community design showcase
- Most viewed and most bought items on the homepage

---

## How to Run

1. Open `index.html` in a web browser  
   **or**
2. Visit the live site using GitHub Pages
